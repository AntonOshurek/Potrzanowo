## what you needs to be done to start the project?

* **download repository**

* **instal Node.js LTS version**
[Download Node.js (LTS)](https://nodejs.org/en/)

* **instal all dependencies**

```shell
npm instal
```

* **Run Dev server**

```shell
npm run start
```
